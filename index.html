Proyectos

Pokédex

Proyecto donde se consume una API pública de pokemon, donde se muestra todos los datos de cada pokemon incluyendo sus tipos y estadisticas, si se accede en la carta de un pokemon se observa una información mas detallada como altura, peso, habilidades pasivas, sus estadisticas respecto a las maximas existentes y algunos de sus posibles movimientos en combate. Realizado con React y estilado en CSS.

Rick and Morty

Proyecto donde se consume la API pública de Rick and Morty, donde se muestran los datos de las diferentes dimensiones incluyendo la población, la especie, el origen y el estado vital del personaje.

Aplicación estado del clima

Aplicación donde se consume una API que nos trae informacion del clima en gran cantidad de regiones a nivel global.

Galleta de la fortuna

Proyecto donde se trabajo la componetizacion de react usando fondos y frases pre-establecidas.

Booking app de Hoteles

Proyecto en el cual se consume una API en la cual se pueden hacer registros y reservaciones en hoteles alrededor del mundo teniendo en cuenta los precios y el tiempo de estadía, asi como un seccion de calificacion y comentarios.
